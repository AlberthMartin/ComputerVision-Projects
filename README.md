Solutions for assignment 1 and assignment 2 in computer vision course.

## Overview

This repository contains coursework assignments for a computer vision course, implemented primarily in Jupyter Notebooks with Python support scripts.

# Structure
- `assigment1.ipynb` Notebook for assignment 1
- `assigment2.ipynb` Notebook for assingment 2

- `project2_functions` python Functions used for assingment 2
- `data`  contains the data used in the assignments
